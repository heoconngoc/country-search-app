# 🌍 Country Search App

A simple web application that allows you to search for information about any country in the world.
Data is sourced from the **REST Countries** API, and maps are displayed using **Google Maps Embed**.

---

## ✨ Features
- Search for countries by name
- Display:
  - National flag
  - Official name
  - Population
  - Region
- Display a Google Maps view of the country
- Simple, easy-to-use interface

---

## 🛠️ Technology
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **REST Countries API**
- **Google Maps Embed**
